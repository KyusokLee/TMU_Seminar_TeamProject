# TMU 朝香ラボのゼミ研究_IoT防災ヘルメットの実装

## 📮 概要

本プロジェクトは朝香ラボのゼミ研究であり、ラズベリーパイとiOSモバイルアプリ、そして、Firebaseを用いて災害地と避難所の共有を目的としたプロジェクトである。
